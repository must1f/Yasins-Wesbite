# Apprenticeship Portal Website

A dual-portal web application connecting employers with apprenticeship candidates.

## Project Overview

This platform provides two distinct portals:
- **Employer Portal**: Post and manage apprenticeship listings with custom application forms
- **Applicant Portal**: Register, upload CVs, and apply for apprenticeships

## Technology Stack

- **Frontend**: HTML5, CSS3, JavaScript, Tailwind CSS
- **Backend**: PHP 8+
- **Database**: MySQL 8
- **Server**: Apache (34SP Hosting)
- **Deployment**: FileZilla FTP

## Documentation

For complete technical specifications, architecture details, and implementation guidelines, see [PRODUCTION_DOCUMENT.md](./PRODUCTION_DOCUMENT.md).

## Key Features

### For Applicants
- User registration and authentication
- Profile management with CV upload
- Browse and filter apprenticeship listings
- Apply through dynamic, employer-defined forms
- Track application status

### For Employers
- Post apprenticeship opportunities
- Create custom application forms with dynamic fields
- Review applications and download CVs
- Shortlist or reject candidates
- Manage active listings
